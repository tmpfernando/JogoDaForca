Acanguçu~Dica: Do tupi akãgu'su que significa "cabeça grande".
Acuty-yauá-retê~Dica: Em Tupi sginifica "Jaguaretê (cor de) cutia".
Anguê~Dica: Palavra de origem africana. "Anguê i cuartá ô orocogombe, isto é, a onça pega o boi".
Anguncê-cuatá~Dica: Palavra de origem africana.
Camisa-de-chita~Dica: Designação dada por caçadores (MT) à onça-pintada.
Camisa-pintada~Dica: Designação usada pelos caçadores (MT) para uma variedade da onça-pinta-da.
Canguçu~Dica: Deriva da palavra indígena Caa-guaçu, significando mata grande ou mato grosso.
Canguçu-açu~Dica: Deriva da palavra indígena "Caa-guaçu", em composição com a palavra "Açú".
Canguçu-de-malha-larga~Dica: Composição derivada da palavra indígena "Caa-guaçu" com uma descrição grandes marcas na pele do animal.
Canguçu-de-malha-miúda~Dica: Composição derivada da palavra indígena "Caa-guaçu" com uma descrição pequenas marcas na pele do animal.
Canguçu-dos-cerrados~Dica: Composição derivada da palavra indígena "Caa-guaçu" com uma descrição da origem do animal.
Canguçu-preto~Dica: Composição derivada da palavra indígena "Caa-guaçu" com uma descrição da coloração mais escura da pele.
Canguçuzinho-do-campo~Dica: Composição derivada da palavra indígena "Caa-guaçu" apresentada no diminutivo, com uma descrição de origem do animal.
Caúba~Dica: Palavra de origem africana. Também é o nome popular de um peixe marinho, o mesmo que guaiúba.
Jaguar~Dica: Mesmo nome de uma marca de carros de luxo.
Jaguaracanguçu~Dica: Composição feita a partir da denominação "Jaguar" com a palavra derivada da denominação indígena "Caa-guaçu".
Jaguapará~Dica: Composição de uma denominação diferente para "onça" com a palavra "pará (pintado)".
Jaguaraí~Dica: Composição de uma denominação diferente para "onça" com a palavra "i (pequeno)".
Jaguarana~Dica: Composição de uma denominação diferente para "onça" com a palavra "rana (falso)".
Jaguarana-pixuna~Dica: Composição de uma denominação diferente para "onça" com as palavras "rana (falso)" e "pixuna (preto)".
Jaguarapara~Dica: Composição de uma denominação diferente para "onça" com a palavra "apara (vergado, curvo)".
Jaguarapinima~Dica: Composição de uma denominação diferente para "onça" com a palavra "pinima (pintado)".
Jaguara-sororoca~Dica: Composição de uma denominação diferente para "onça" com uma palavra que também é o nome de um peixe que no nordeste do Brasil como "serra".
Jaguaraúna~Dica: Composição de uma denominação diferente para "onça" com uma palavra "una (preta)".
Jaguaretê~Dica: Composição de uma denominação diferente para "onça" com uma palavra "etê (verdadeiro)".
Jaguaretê-apara~Dica: Composição de uma denominação diferente para "onça" com "para (multicor)".
